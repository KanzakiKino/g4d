g4d
----

g4d is a graphics library for dlang.

**This is developing now.**

## Requirements

- [libeasyff](https://github.com/KanzakiKino/easyFF) 0.0.2-alpha
- Freetype2 2.9.1
- GLFW 3.2.1 or more
- OpenGL 3.3 or more

You can see `/test.d`.
