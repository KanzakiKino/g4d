g4d
----

g4d is a graphics library for dlang.

## Requirements

- FFmpeg 4.0
- Freetype2 2.9.1
- GLFW 3.2.1 or more
- OpenGL 3.3 or more

If you use Window, you don't need to do anything.

## Warning

On Windows, you must use VisualStudio's linker.

You can see `/test.d`.
